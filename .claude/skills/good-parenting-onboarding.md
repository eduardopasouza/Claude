# Good Parenting — Skill: Onboarding e Atualização de Perfil

## Quando ativar
- Novo usuário (primeira vez usando o plugin)
- Atualização massiva de dados (novo bebê, mudança de cidade, separação, etc.)
- Revisão periódica de perfil (a cada 3-6 meses)

## Para novo usuário
Fazer perguntas uma a uma, em ordem:

### Bloco 1 — O filho
1. Nome e data de nascimento
2. Saúde geral, condições especiais
3. Dados atuais (peso, altura, marcos de desenvolvimento)
4. Alimentação (peito, fórmula, misto, introdução alimentar)
5. Sono (rotina, dificuldades)

### Bloco 2 — O pai
1. Nome, idade, profissão
2. Pai de primeira viagem?
3. Maior preocupação como pai
4. Valores e filosofia de vida
5. Religião/espiritualidade
6. Vícios/hábitos que quer mudar

### Bloco 3 — A família
1. Situação conjugal
2. Dados da mãe/parceira
3. Rede de apoio (avós, babá, creche)
4. Dinâmica do dia a dia
5. Conflitos ou tensões familiares

### Bloco 4 — Criação
1. Linha de criação (disciplina positiva, Montessori, etc.)
2. Posição sobre telas
3. Planos para educação formal
4. Decisões já tomadas (batismo, alimentação, etc.)
5. Referências que segue (livros, perfis, pediatra)

## Ao finalizar
1. Criar/atualizar todos os perfis em `memoria/perfis/`
2. Criar registro da sessão em `memoria/sessoes/`
3. Atualizar `memoria-geral.md`
4. Apresentar resumo ao pai para validação

## Para revisão periódica
1. Leia todos os perfis atuais
2. Pergunte o que mudou em cada área principal
3. Atualize apenas o que mudou
4. Registre a revisão como sessão
