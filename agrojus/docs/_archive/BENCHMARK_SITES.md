# AgroJus — Sites de Referência e Benchmark

Este documento serve como referência de alto nível para UX, features e design de produto. Destinado para o Antigravity, Supra Gerente e UX Designers.

|---|---|---|---|---|

| 2 | SpectraX | Compliance MCR 2.9 / EUDR via satélite + IA. Deep tech de Sinop/MT | Dashboard de compliance, checklist MCR, visual de score | [spectrax.com.br](https://spectrax.com.br) |

| 4 | AdvLabs | IA para defesa ambiental (IBAMA/ICMBio). Teses e petições | Interface de busca jurídica, fluxo de defesa ambiental | [advlabs.com.br](https://advlabs.com.br) |
| 5 | Docket | OCR de certidões com IA (50+ tipos). Parceira da Agrotools | UX de upload de documento, timeline de certidões | [docket.com.br](https://docket.com.br) |
| 6 | Serasa Agro | Agro Score, compliance MCR, monitoramento via satélite | Score visual (gauge chart), alertas, semáforo de risco | [serasaexperian.com.br/agronegocio](https://serasaexperian.com.br/agronegocio) |
