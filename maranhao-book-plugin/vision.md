# Vision — Quem é o Maranhão?

> Premissas editoriais, decisões de design e expectativas do almanaque.

## A pergunta

"Quem é o Maranhão?" não é retórica. É uma pergunta genuína que 106 verbetes tentam responder, cada um de um ângulo — e que permanece aberta no epílogo.

## Premissas editoriais

### 1. Dados + Alma
Todo número conta uma história humana. Todo parágrafo narrativo tem sustentação factual. A alma não substitui os dados; os dados não substituem a alma. Os dois coexistem em cada frase.

### 2. Três matrizes com peso igual
Indígena, africana e europeia/migrante recebem tratamento idêntico em profundidade, espaço e qualidade de pesquisa. Isso é raro em qualquer livro regional brasileiro. É o diferencial ético do projeto.

### 3. Narrador maranhense implícito
O livro não explica o Maranhão de fora. Narra de dentro — com "nossa terra", "nosso povo" — mas sem localismo excludente. O leitor de fora é bem-vindo; o leitor de dentro se reconhece.

### 4. Rigor sem academicismo
Fontes verificadas, notas de rodapé, banco de dados central. Mas a linguagem é de crônica e ensaio, não de tese. O modelo é Eduardo Bueno e Eliane Brum, não um paper do SciELO.

### 5. Ironia com história, neutralidade com o presente
Eventos históricos podem receber tratamento irônico (contradições coloniais, absurdos do império). O presente recebe dados frios — o leitor tira suas conclusões.

### 6. O livro não é denúncia, lamento, panfleto, guia turístico nem livro didático
É um retrato em mosaico. Mostra o que é, com dados e contexto. Não propõe soluções, não acusa nomes, não romantiza a pobreza.

## Decisões de design

### Formato
- Almanaque visual, 23×28cm (coffee table book)
- Primeira metade cronológica (Partes I–V), segunda metade temática (Partes VI–XI)
- Dentro de cada Parte, ordem livre — leitor entra por qualquer verbete
- Tamanho dos verbetes: livre (500 a 5.000+ palavras)

### Identidade visual
- Paleta: Ocre, Terracota, Verde-mata, Azul-mar (primárias) + Areia, Creme, Carvão, Vermelho-bumba, Roxo-tambor (secundárias)
- Tipografia: Playfair Display (títulos) + Source Serif Pro (texto) + Source Sans Pro (boxes/dados)
- Fundo: Creme (#FAF3E8) para livro, Carvão (#2B2B2B) para vídeo — nunca branco puro
- Cada Parte tem cor de acento própria (ver `style-guide-visual.md`)

### Elementos narrativos
- **8 séries de boxes**: Lendas, Maranhenses, E no Brasil, Enquanto Isso, Receitas, Glossário, Documento, Contrafactual
- **Easter eggs**: Fio da água (referências orgânicas a água), Naufrágios (metáfora recorrente)
- **9 apêndices**: Cronologia, Glossário, Índice, Bibliografia, TIs, Quilombos, Municípios, Recordes, Biográfico

### Multiplataforma
- Cada verbete gera obrigatoriamente: texto + reels + briefing visual
- YouTube: apenas para verbetes fortes (Claude sugere no outline)
- Site companion: verbetes expandidos + dados atualizados
- Documentário: produto futuro

- ISBN: ao final da produção

## Critérios de qualidade

Um verbete está pronto quando:
1. Toda afirmação factual tem ≥3 fontes verificadas
2. Todo dado está registrado no banco YAML
3. O tom é consistente com o epílogo-bússola
4. As 3 matrizes estão representadas (quando aplicável)
5. Cross-references estão mapeados
6. Reels e visual estão produzidos
7. O autor aprovou
