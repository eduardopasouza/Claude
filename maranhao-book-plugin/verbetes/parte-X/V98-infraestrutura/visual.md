# Visual V98-A: O que tem vs. o que falta

## 1. Briefing para designer
**O que e**: Infografico dividido ao meio. Lado esquerdo (dourado): infraestrutura que existe (ferrovias, porto, energia). Lado direito (cinza): infraestrutura que falta (esgoto, agua, moradia, internet).
**Paleta**: Dourado/ocre para "tem", carvao/cinza para "falta".
**Tamanho**: Meia pagina.
