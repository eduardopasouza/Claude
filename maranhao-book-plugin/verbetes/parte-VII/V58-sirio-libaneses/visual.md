# Visual V58-A: A rota do mascate

## 1. Briefing para designer
**O que é**: Infográfico de rota migratória sírio-libanesa até o Maranhão, com timeline de ascensão social.
**Dados**:
- Rota: Beirute → Marselha → Belém → São Luís / interior do MA
- Timeline de gerações:
  - Geração 1 (1880-1910): Mascate ambulante — ícone de mala + mula
  - Geração 2 (1910-1950): Lojista fixo — ícone de fachada de loja
  - Geração 3 (1950-2000): Político/profissional liberal — ícone de palanque
- Sobrenomes marcantes: Sarnehy→Sarney, Dino, Haickel, Murad
- Ícone de quibe frito como elemento visual de fusão cultural
**Referência de estilo**: Kurzgesagt — ícones simplificados, setas curvas sobre mapa
**Paleta**: #B5533E (Terracota Parte VII) + #006600 (cedro libanês) + #FAF3E8 (creme fundo)
**Tamanho**: Terço de página (23×9cm)
