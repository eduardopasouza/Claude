# Visual V69-A: Anatomia de uma radiola

## 1. Briefing para designer
**O que é**: Infográfico detalhando a estrutura de uma radiola maranhense — paredão de caixas de som, amplificadores, vitrola, seletor. Comparação com sound system jamaicano. Página inteira.
**Elementos**: Diagrama lateral de um paredão, com setas indicando caixas, amplificadores, mesa do seletor, vitrola, discos. Dados: 200+ radiolas no MA, batalhas de som, pedras secretas.
**Paleta**: Verde-Jamaica (#009B3A), Dourado (#D4AC0D), Preto (#1C1C1C)
**Tamanho**: Página inteira
