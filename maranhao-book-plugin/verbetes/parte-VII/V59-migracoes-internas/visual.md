# Visual V59-A: Mapa dos fluxos migratórios internos

## 1. Briefing para designer
**O que é**: Mapa do Maranhão com setas de fluxos migratórios internos (de fora e de dentro), codificados por cor e época.
**Dados**:
- Seta 1 — Ceará/Piauí → Leste do MA: cor Azul #1B4965, séc. XVII-XX (frente pastoril + retirantes)
- Seta 2 — Bahia → Sul do MA: cor Ocre #C8952E, séc. XVIII-XIX (gado + cerrado)
- Seta 3 — Goiás/Minas → Sul do MA: cor Vermelho #C1292E, séc. XX (fronteira agrícola + soja)
- Seta 4 — Interior → São Luís: cor Laranja #E87C3E, séc. XX-XXI (êxodo rural)
- Cidades-polo: Imperatriz, Caxias, Balsas, São Luís (com população em datas-chave)
- Ícones por fluxo: vaqueiro (CE/BA), trator (GO/MG), ônibus (rural→SLZ)
- Rodovia Belém-Brasília em tracejado como eixo de penetração
**Referência de estilo**: Kurzgesagt — setas curvas, ícones simplificados
**Paleta**: #B5533E (Terracota Parte VII) + cores dos fluxos acima + #FAF3E8 (creme)
**Tamanho**: Página inteira (23×28cm)
