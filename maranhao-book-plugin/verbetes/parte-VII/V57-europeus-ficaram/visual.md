# Visual V57-A: Mapa das três ocupações europeias

## 1. Briefing para designer
**O que é**: Mapa cronológico de São Luís mostrando as três camadas de ocupação europeia (1612-1644) com setas de origem transatlântica.
**Dados**:
- Camada 1 — França (1612-1615): seta de La Rochelle a São Luís, cor Azul-França #002654
- Camada 2 — Holanda (1641-1644): seta de Amsterdã a São Luís, cor Laranja-WIC #FF6B00
- Camada 3 — Portugal (1615→permanente): seta de Lisboa a São Luís, cor Verde-Portugal #006600
- Timeline no rodapé: 1612 | 1615 | 1641 | 1644 | 1755 | 1997 (UNESCO)
- Detalhes no mapa de São Luís: Forte de Santo Antônio, Palácio dos Leões, Centro Histórico
**Referência de estilo**: Kurzgesagt — flechas curvas sobre Atlântico, cores vibrantes
**Paleta**: #002654 (França) + #FF6B00 (Holanda) + #006600 (Portugal) + #B5533E (acento Terracota da Parte VII)
**Tamanho**: Meia página (23×14cm)

## 2. Briefing visual secundário

**VIS-V57-B**: Grid fotográfico 3×2 de azulejos
- 6 padrões diferentes de azulejos de fachada de São Luís
- Cada célula: 1 padrão com legenda (época estimada, estilo)
- Tamanho: terço de página (23×9cm)
- Referência: catálogo IPHAN / Andrès (1998)

## 3. Notas para diagramação
- Accent da Parte VII: Terracota #B5533E
- Título do verbete em Terracota sobre fundo Creme #FAF3E8
- Boxes com borda Terracota, fundo Creme
