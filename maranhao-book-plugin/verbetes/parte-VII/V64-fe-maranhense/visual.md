# Visual V64-A: As camadas da fé maranhense

## 1. Briefing para designer
**O que é**: Diagrama de camadas sobrepostas mostrando as tradições religiosas do Maranhão e suas interseções.
**Dados**:
- Camada base (mais antiga): Pajelança indígena — cor Verde-mata #2D6A4F
- Camada 2: Catolicismo popular — cor Azul #1B4965
- Camada 3: Tambor de Mina / Voduns — cor Vermelho-escuro #8B0000
- Camada 4: Terecô — cor Roxo #6B3FA0
- Camada 5 (recente): Evangélicos — cor Cinza #666666
- Zona de interseção central: ENCANTADOS — cor Dourado #D4A017
- Setas de influência mútua entre camadas
- Ícones: terço (catolicismo), tambor (Mina), ervas (pajelança), Bíblia (evangélicos)
**Referência de estilo**: Diagrama de Venn estilizado / camadas geológicas
**Paleta**: #B5533E (Terracota Parte VII) + cores acima + #FAF3E8 (creme)
**Tamanho**: Página inteira (23×28cm)

## 2. Visual secundário: VIS-V64-B
**O que é**: Mapa de São Luís com localização dos terreiros históricos
- Casa das Minas (Querebentã)
- Casa de Nagô
- Terreiros do bairro da Madre Deus
- Igrejas históricas (Catedral, São Benedito)
- Overlay: raio de 2km mostrando coexistência terreiro/igreja
- Tamanho: terço de página (23×9cm)

## 3. Notas para diagramação
- Accent da Parte VII: Terracota #B5533E
- Verbete-âncora: merece spread duplo
- Três boxes: espaçar ao longo do texto para não sobrecarregar uma página
- Tons quentes no visual — evitar azul frio
