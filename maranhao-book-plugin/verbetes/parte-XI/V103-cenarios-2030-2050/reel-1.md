# Reel V103-1: O padrão que se repete — e a pergunta

## HOOK (0-3s)
**TELA**: "Algodão. Minério. Soja. O próximo ciclo vai ser diferente?"

## DESENVOLVIMENTO (3-45s)
**NARRAÇÃO**: "No século XVIII, o Maranhão ficou rico com o algodão. A riqueza saiu. No século XX, o minério de Carajás passou pela ferrovia — 177 milhões de toneladas por ano. Os royalties foram para o Pará."

"No século XXI, a soja transformou o cerrado do sul do estado. O PIB per capita continua o menor do Brasil."

"Agora o Maranhão é candidato a hub de energias renováveis. Tem vento, sol, porto de águas profundas, potencial de hidrogênio verde. Tudo que precisa para diversificar."

"A pergunta é: vai repetir o padrão — riqueza que passa sem ficar — ou vai quebrá-lo?"

**TELA**: "O Maranhão tem os ativos. A pergunta é o que faz com eles."
