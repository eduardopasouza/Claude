## Relevâncias globais do Maranhao

| Dimensao | Relevância |
|----------|-----------|
| Biodiversidade | 4 biomas, Amazonia+Cerrado+Caatinga+Costeiro |
| Mangues | Maior area continua das Americas (Reentrâncias) |
| Minerio de ferro | Rota de exportacao da maior jazida do planeta |
| Soja | Hub de exportacao do Arco Norte (alimenta China) |
| Cultura | Bumba-meu-boi (Patrimônio Imaterial UNESCO), reggae |
| Patrimônio | Centro Histórico de São Luís (Patrimônio Mundial UNESCO) |
| Energia | Potencial eólico e solar de escala global |
| Carbono | Estoque de carbono em Amazônia e cerrado maranhense |
| Lençóis Maranhenses | Formação geológica única no planeta |
| Parcel Manuel Luís | Maior banco de corais do Atlântico Sul |
