# Visual V14-A: Mapa das Terras Indígenas do Maranhão

## 1. Briefing para designer
**O que é**: Mapa do Maranhão mostrando todas as Terras Indígenas, com sobreposição da cobertura florestal remanescente. Visual principal — mostra que as últimas florestas coincidem com os territórios indígenas.
**Dados**: ~20 TIs marcadas (Alto Turiaçu 530.524 ha, Arariboia 413.000 ha, Porquinhos 301.000 ha, Krikati 159.169 ha, Bacurizinho 134.040 ha, Canela 125.212 ha, Caru ~172.000 ha, Governador 41.642 ha, etc.). Cobertura florestal em verde sobreposta. Zonas de conflito marcadas.
**Paleta**: Verde-floresta (#1E8449) para TIs com floresta, Vermelho-alerta (#C0392B) para zonas de conflito, Bege (#F5F5DC) para áreas desmatadas
**Tamanho**: Página inteira

## 2. Prompt para IA generativa
**Prompt**: "Map of Maranhão state, Brazil, showing Indigenous Territories as green patches overlapping with remaining forest cover. The contrast should be stark: areas outside Indigenous Lands are mostly deforested (beige/brown), while areas inside are green forest. Mark key territories by name. Red markers for conflict zones (Arariboia, Alto Turiaçu). Clean cartographic style, educational."

# Visual V14-B: De 250.000 a 57.214 — a população indígena do Maranhão

## 1. Briefing para designer
**O que é**: Infográfico mostrando o colapso demográfico — de 250.000 (pré-contato) a 57.214 (Censo 2022). Timeline com marcos: contato, aldeamentos jesuítas, Pombal, Ferrovia Carajás, recuperação recente.
**Paleta**: Verde-terra, preto para dados de perda, dourado para marcos de resistência
**Tamanho**: Meia página

# Visual V14-C: Os povos originários — diversidade étnica e linguística

## 1. Briefing para designer
**O que é**: Infográfico mostrando os ~17 povos indígenas atuais do MA, agrupados por tronco linguístico (Tupi-Guarani vs. Macro-Jê/Timbira), com população e status. Formato de árvore ou mapa mental.
**Dados**: Guajajara (27.000+), Ka'apor (~2.000), Awá (~350), Canela Ramkokamekrá (~2.000), Canela Apãniekrá, Krikati (~1.000+), Gavião (~500+), Krepumkateyê, Krenye, Tremembé, Akroá-Gamela, etc.
**Paleta**: Duas cores de base (uma para Tupi, uma para Jê), com tamanho proporcional à população
**Tamanho**: Corpo de texto
