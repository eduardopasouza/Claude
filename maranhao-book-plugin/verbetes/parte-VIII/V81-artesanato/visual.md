# Visual V81-A: Grid de artesanato maranhense
## Briefing para designer
Grid 3x2 com fotos/ilustracoes de artesanato maranhense:
1. Ceramica de Itamatatiua — potes e panelas de barro, maos femininas negras
2. Renda de bilro — almofada com bilros e renda branca em progresso
3. Trancado de buriti — cestos, esteiras, bolsas de fibra dourada
4. Produtos de babacu — sabao, oleo em garrafas, farinha de mesocarpo
5. Bordado de couro de boi — couro com micancas, canutilhos, lantejoulas
6. Cestaria de palha — chapeus, abanos, cestos decorativos
Cada celula com legenda: tecnica, origem cultural (africana/portuguesa/indigena/maranhense), regiao.
Paleta: #C1292E (Vermelho-bumba) + cores naturais (barro, palha, branco) + #FAF3E8 (creme).
Tamanho: Pagina inteira (23x28cm).
