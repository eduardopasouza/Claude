# Outline — V77: Josué Montello
## Tese: Montello é o escritor que transformou São Luís em personagem. Cada romance é mapa sentimental da cidade. Os Tambores de São Luís conta a saga negra maranhense com a ambição de um Balzac tropical.
## Estrutura
### 1. Abertura (~300): São Luís como romance
### 2. Os Tambores de São Luís (~500): saga de família negra
### 3. O cronista da memória (~500): diários, ABL, vida no Rio
### 4. Fechamento (~300): Nunca saiu de São Luís (cross-refs: V41, V50)
