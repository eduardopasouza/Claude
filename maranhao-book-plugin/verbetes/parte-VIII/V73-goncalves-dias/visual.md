# Visual V73-A: A vida de Gonçalves Dias

## 1. Briefing para designer
**O que é**: Timeline horizontal da vida de Gonçalves Dias com marcos biográficos.
**Dados**:
- 1823: Nasce em Caxias (ícone: palmeira)
- 1840: Parte para Coimbra (ícone: navio)
- 1843: Escreve "Canção do Exílio" (ícone: pena)
- 1847: Primeiros Cantos — sucesso (ícone: livro)
- ~1852: Ana Amélia — recusa (ícone: coração partido)
- 1851: I-Juca-Pirama (ícone: arco indígena)
- 1864: Naufrágio — morte (ícone: navio afundando)
**Estilo**: Timeline minimalista com ícones e datas, retrato central de GD
**Paleta**: #C1292E (Vermelho-bumba Parte VIII) + sépia + #FAF3E8 (creme)
**Tamanho**: Meia página (23×14cm)

## 2. Notas para diagramação
- Accent da Parte VIII: Vermelho-bumba #C1292E
