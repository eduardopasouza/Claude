# Visual V46-A: São Luís × São Paulo — curvas divergentes (1872-1960)

## 1. Briefing para designer
**O que é**: Gráfico comparativo de duas curvas populacionais mostrando a divergência radical entre São Luís e São Paulo ao longo de ~90 anos.
**Dados**:
- 1872: São Luís ~31.000 / São Paulo ~31.000 (ponto de partida IGUAL)
- 1890: São Luís ~35.000 / São Paulo ~65.000
- 1900: São Luís ~38.000 / São Paulo ~240.000
- 1920: São Luís ~52.000 / São Paulo ~580.000
- 1940: São Luís ~85.000 / São Paulo ~1.300.000
- 1960: São Luís ~160.000 / São Paulo ~3.800.000
- Anotações: "Café reinvestido" (SP, 1880s), "Algodão exportado" (SL, 1870s), "Ferrovias" (SP, 1867), "Nenhuma ferrovia" (SL), "USP fundada" (SP, 1934), "UFMA fundada" (SL, 1966)
**Referência**: Gráficos de divergência estilo Our World in Data / Economist
**Paleta**: Ocre (#C8952E) para linha do MA, azul escuro para SP, fundo creme, anotações em carvão
**Tamanho**: Página inteira
**Impacto**: O gráfico sozinho conta a história. As duas linhas começam juntas e se separam como tesoura. A distância visual É a tese.

## 2. SVG simplificado
**Elementos**: Eixo X (anos: 1872-1960), eixo Y (população), duas linhas que começam no mesmo ponto e divergem drasticamente. SP sobe em exponencial; SL mal se move. Anotações nos pontos de inflexão.
**Layout**: Horizontal, clean, com espaço para as anotações não se sobreporem.

## 3. Prompt para IA generativa
**Prompt**: "Dual-line chart infographic comparing population growth of São Luís and São Paulo, Brazil, 1872-1960. Both cities start at identical population (~31,000) in 1872 and dramatically diverge: São Paulo rockets to 3.8 million by 1960 while São Luís barely reaches 160,000. Annotated inflection points showing 'coffee reinvested', 'railways built', 'university founded' for SP, and 'cotton exported', 'no railways', 'no university until 1966' for SL. Warm ochre accent for Maranhão line, dark navy for SP. Cream background. Clean editorial style like Economist or Our World in Data. Title: 'Same starting point. Different choices.'"
**NÃO incluir**: 3D effects, gradients desnecessários, elementos decorativos

# Visual V46-B: O século esquecido — linha do tempo (1850-1966)

## 1. Briefing para designer
**O que é**: Linha do tempo horizontal cobrindo 116 anos, mostrando os marcos (poucos) e o VAZIO (protagonista) do período.
**Dados**:
- 1850: Declínio definitivo do algodão inicia
- 1861-65: Breve boom (Guerra Civil Americana)
- 1867: São Paulo inaugura ferrovia Santos-Jundiaí (contraste)
- 1872: Censo — SL e SP com ~31.000 hab.
- 1888: Abolição — fazendas desmoronam
- 1889: República — coronelismo se instala
- 1900: São Luís estagnada; SP em explosão industrial
- 1920: Censo — SP 580.000 / SL 52.000
- 1930: Era Vargas — coronelismo se adapta
- 1933: Vitorino Freire chega ao MA
- 1934: USP fundada (contraste)
- 1945-65: Vitorino domina a política maranhense
- 1950: Analfabetismo >70% no MA
- 1965: Sarney derrota Vitorino
- 1966: UFMA fundada — 139 anos depois da faculdade de Direito de SP
**Conceito visual**: A linha do MA é quase vazia — poucos eventos, muito espaço branco. A linha de SP (abaixo, em contraste) é cheia de marcos. O vazio É a mensagem.
**Referência**: Timelines do NYT Visual Journalism
**Paleta**: Ocre para marcos do MA, azul escuro para marcos de SP (contraste), fundo creme
**Tamanho**: Spread (duas páginas) ou dobrável

## 2. SVG simplificado
**Elementos**: Duas linhas horizontais paralelas (MA e SP). Marcos como pontos com rótulo. A linha do MA tem enormes espaços vazios entre marcos. A linha de SP é densa.
**Layout**: Horizontal, leitura esquerda→direita, anos no eixo superior.

## 3. Prompt para IA generativa
**Prompt**: "Dual horizontal timeline comparing Maranhão and São Paulo, Brazil, 1850-1966. Top line (Maranhão, ochre) is nearly EMPTY — very few milestones, vast whitespace between events, showing stagnation. Bottom line (São Paulo, navy) is DENSE with milestones — railways, factories, universities, immigration waves. Key contrast points connected by vertical dotted lines (e.g., 1872 census showing same population). Cream background, editorial style, clean typography. The visual emptiness of the Maranhão line IS the message."
**NÃO incluir**: Decoração, ícones infantis, clipart

# Visual V46-C: O que não foi construído — ferrovias SP × MA (c. 1920)

## 1. Briefing para designer
**O que é**: Mapa comparativo lado a lado: São Paulo com sua rede ferroviária densa (c. 1920) e Maranhão completamente vazio no mesmo período.
**Dados**:
- SP (1920): Rede ferroviária densa — SPR, Mogiana, Paulista, Sorocabana, Araraquarense. Centenas de km.
- MA (1920): Nenhuma ferrovia relevante. Zero km.
- Legenda: "O que o café construiu" (SP) / "O que o algodão não deixou" (MA)
**Referência**: Mapas ferroviários históricos sobrepostos a silhuetas de estado
**Paleta**: Ocre para MA (silhueta vazia), azul escuro para SP (linhas ferroviárias densas), fundo creme
**Tamanho**: Meia página, dois mapas lado a lado

## 2. SVG simplificado
**Elementos**: Dois contornos de estado lado a lado. SP cheio de linhas (ferrovias). MA vazio. Legenda embaixo.
**Layout**: Horizontal, dois mapas, mesma escala se possível (MA é maior em área, o que torna o vazio ainda mais dramático).

## 3. Prompt para IA generativa
**Prompt**: "Side-by-side map comparison: São Paulo state (left) with dense railway network circa 1920, showing multiple lines crisscrossing the territory, versus Maranhão state (right) completely empty — no railways at all. SP map filled with navy-blue rail lines. MA map is just an empty ochre outline. Caption: 'What coffee built' (SP) / 'What cotton didn't leave behind' (MA). Cream background, clean cartographic style. The emptiness of Maranhão's map is the point."
**NÃO incluir**: Fotos, 3D terrain, decoração desnecessária
