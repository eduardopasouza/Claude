# Visual V47-A: A maquina vitorinista

## 1. Briefing para designer
**O que e**: Diagrama circular mostrando o ciclo do clientelismo vitorinista. Meia pagina.
**Dados/fluxo**:
- VITORINO (centro) → Indica cargos publicos
- Cargos publicos → Geram lealdade
- Lealdade → Produz votos
- Votos → Elegem candidatos de Vitorino
- Candidatos eleitos → Fazem mais nomeacoes
- (volta ao centro)
**Referencia de estilo**: Diagrama editorial circular, tipo McKinsey simplificado
**Paleta**: Ocre #C8952E, Cinza-chumbo #4A4A4A, Branco
**Tamanho**: Meia pagina (23x14 cm)

## 2. Elementos visuais
- Circulo central: retrato estilizado de Vitorino
- 5 nos ao redor conectados por setas
- Cada no com icone: selo (cargo), mao (lealdade), urna (voto), cadeira (candidato), carimbo (nomeacao)
- Texto curto em cada no

# Visual V47-B: Timeline Vitorino → Sarney

## 1. Briefing para designer
**O que e**: Timeline horizontal mostrando a transicao de poder Vitorino → Sarney. Coluna.
**Dados**:
- 1930s: Vitorino chega ao MA
- 1945: Redemocratizacao, Vitorino domina PSD
- 1947-1971: Vitorino senador
- 1950s: Sarney surge como afilhado politico
- 1965: Ruptura Sarney-Vitorino
- 1966: Sarney governador ("Maranhao Novo")
- 1977: Morte de Vitorino
- 1985-2010s: Era Sarney (governador → senador → presidente → familia)
**Paleta**: Ocre, cinza, vermelho para a ruptura
**Tamanho**: Coluna lateral (8x28 cm)
