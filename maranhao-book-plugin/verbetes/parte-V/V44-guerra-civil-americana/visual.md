# Visual V44-A: O preco do algodao e as guerras

## 1. Briefing para designer
**O que e**: Grafico de linha mostrando o preco do algodao em Liverpool (1780-1880) com faixas destacando os dois booms maranhenses. Meia pagina.
**Dados**:
- Eixo X: 1780-1880
- Eixo Y: pence por libra (Liverpool)
- Primeiro boom: ~1780-1820 (Revolucao Americana + Napoleonicas)
- Vale: 1820-1860 (algodao americano domina)
- Segundo boom: 1861-1865 (Guerra Civil Americana)
- Pico 1864: ~27 pence/libra
- Queda pos-1865: volta a ~7-8 pence
**Referencia de estilo**: Grafico editorial tipo Economist — limpo, poucos elementos, dados claros
**Paleta**: Ocre #C8952E para booms, Cinza #888 para periodos normais, Vermelho para guerras
**Tamanho**: Meia pagina (23x14 cm)

## 2. Elementos visuais
- Linha de preco com areas sombreadas nos booms
- Faixas verticais vermelhas semi-transparentes para guerras
- Anotacoes: "Rev. Americana", "Napoleonicas", "Guerra Civil EUA"
- Setas descendentes pos-boom com texto "O MA perde"
