# Visual V29-A: Os azulejos de São Luís

## 1. Briefing para designer
**O que é**: Composição visual de azulejos portugueses de São Luís — grid de 9 padrões diferentes (3x3) representando os estilos encontrados no centro histórico: barrocos, pombalinos, neoclássicos e industriais. Página inteira.
**Dados**: 9 padrões reais de azulejos de São Luís (referência: acervo IPHAN)
**Paleta**: Azul-mar (#1B4965), Branco, Amarelo, Verde — cores tradicionais dos azulejos
**Tamanho**: Página inteira

## 2. Prompt para IA generativa
**Prompt**: "Grid of 9 different Portuguese azulejo tile patterns (3x3) found in São Luís, Maranhão, Brazil. Styles include: Baroque (ornate blue-white), Pombaline (geometric blue-white), neoclassical (floral motifs), and industrial 19th century (polychrome). Each tile pattern in a square. White grout lines between them. High detail, photorealistic ceramic texture. Color palette: traditional blues, whites, yellows, greens of Portuguese tiles. Clean presentation, museum-quality documentation."

# Visual V29-B: De onde vieram os portugueses do Maranhão

## 1. Briefing para designer
**O que é**: Mapa de Portugal com setas indicando as regiões de origem dos colonos que vieram ao Maranhão: Açores, Minho, Trás-os-Montes, Madeira, Lisboa. Meia página.
**Dados**:
- Açores (seta grossa): maior contingente, séc. XVIII
- Minho: comerciantes, artesãos
- Trás-os-Montes: agricultores
- Lisboa: funcionários, militares
- Madeira: menor contingente
**Paleta**: Azul-mar (#1B4965), Verde (#1E8449) para Portugal, Dourado (#D4AC0D) para setas
**Tamanho**: Meia página

## 2. Prompt para IA generativa
**Prompt**: "Map of Portugal and the Azores islands showing migration routes to Maranhão, Brazil. Golden arrows from: Azores (thick arrow, major migration), Minho region (medium), Trás-os-Montes (thin), Lisbon (medium), Madeira (thin). Destination: São Luís, Maranhão shown as inset. Clean cartographic style, soft colors, educational."
