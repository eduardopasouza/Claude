# Reel V29-1: A herança que ninguém nomeia

## HOOK (0-3s)
**TELA**: "Tudo o que você vê no centro de São Luís é português. Mas ninguém diz isso."

## DESENVOLVIMENTO (3-40s)
**NARRAÇÃO**: "Os azulejos. Os sobrados. As igrejas barrocas. Os nomes das ruas. A língua que você usa para ler as placas. Tudo isso é herança portuguesa."

"São Luís tem o maior acervo de azulejos portugueses FORA de Portugal. Por isso é Patrimônio da Humanidade."

"Mas ninguém diz 'herança portuguesa' como diz 'herança africana' ou 'herança indígena.' A herança portuguesa virou paisagem. É o ar que você respira sem perceber."

## SURPRESA/CTA (40-60s)
**TELA**: "Três matrizes fizeram o Maranhão: indígena, africana e portuguesa. Nomear as três é o primeiro passo para entender o todo."
**TELA**: "Quem é o Maranhão? #29"
