# Visual V28-A: Antes e depois de Pombal — a transformação do Maranhão

## 1. Briefing para designer
**O que é**: Infográfico de duas colunas — "Antes da Companhia (pré-1755)" vs. "Depois da Companhia (1778)" — mostrando a transformação radical do Maranhão. Página inteira.
**Dados**:
- ANTES: 34 africanos/ano, economia de subsistência, drogas do sertão, população indígena/mestiça, São Luís = vila pobre
- DEPOIS: milhares de africanos/ano, algodão+arroz para exportação, 4ª capitania mais rica, 55% escravizada (1821), São Luís = cidade de azulejos
**Paleta**: Azul-mar (#1B4965), Terracota (#BA4A00) para dados demográficos, Dourado (#D4AC0D) para economia
**Tamanho**: Página inteira

## 2. Prompt para IA generativa
**Prompt**: "Split infographic comparing Maranhão before and after Pombal's Company (1755). Left panel (muted, gray tones): tiny figures representing 34 Africans per year, small huts, subsistence farming. Right panel (vivid colors): thousands of figures, ships, cotton bales, rice sacks, colonial mansions with Portuguese tiles. Center dividing line: '1755 — A Companhia'. Bottom: population bar showing transformation from indigenous/mestizo to 55% enslaved. Clean editorial style."

# Visual V28-B: A rota triangular da Companhia

## 1. Briefing para designer
**O que é**: Mapa do Atlântico mostrando a rota triangular Lisboa → Bissau/Cacheu → São Luís → Lisboa. Meia página.
**Dados**:
- Perna 1: Lisboa → Guiné (tecidos, armas, ferro, aguardente)
- Perna 2: Guiné → São Luís (pessoas escravizadas — 31.317 embarcados)
- Perna 3: São Luís → Lisboa (algodão, arroz, cacau)
- Portos: Lisboa, Bissau, Cacheu, São Luís
**Paleta**: Azul-mar (#1B4965), Setas douradas (#D4AC0D), Pontos terracota (#BA4A00)
**Tamanho**: Meia página

## 2. Prompt para IA generativa
**Prompt**: "Triangular trade route map of Pombal's Company (1755-1778). Atlantic Ocean. Three golden curved arrows forming triangle: Lisbon to Guinea-Bissau (carrying goods), Guinea-Bissau to São Luís Brazil (carrying enslaved people, thick arrow), São Luís to Lisbon (carrying cotton and rice). Ports marked with dots. Ship icons on each leg. Data labels on arrows. Clean modern cartographic style, aged parchment background."
