# Visual V26-A: O Brasil holandês — de Sergipe ao Maranhão

## 1. Briefing para designer
**O que é**: Mapa da costa nordeste do Brasil mostrando a extensão do domínio holandês (WIC) entre 1630 e 1654, com destaque para a extensão ao Maranhão (1641-1644). Meia página.
**Dados**:
- Núcleo: Recife/Olinda (1630-1654) — sede de Nassau
- Extensão sul: Sergipe
- Extensão norte: Ceará, depois Maranhão (1641)
- São Luís e Alcântara marcados como pontos ocupados
- Pernambuco em cor sólida (24 anos), Maranhão em cor mais leve (2 anos)
**Paleta**: Laranja-holandês (#E67E22) para domínio WIC, Azul-mar (#1B4965) para oceano, Verde (#1E8449) para território português
**Tamanho**: Meia página

## 2. Prompt para IA generativa
**Prompt**: "Map of northeastern Brazil coast showing Dutch West India Company (WIC) territorial control, 1630-1654. Orange-shaded territory from Sergipe to Maranhão along the coast. Recife/Olinda marked as capital with star. São Luís marked as northern outpost (1641-1644) with lighter shading. Atlantic Ocean in deep blue. Portuguese territory in green. Clean cartographic style, modern infographic aesthetic. Compass rose. Scale bar."

# Visual V26-B: Cronologia das três ocupações de São Luís

## 1. Briefing para designer
**O que é**: Timeline horizontal mostrando as três ocupações coloniais de São Luís: francesa (1612-1615), portuguesa (1615-1641 e 1644+), holandesa (1641-1644). Corpo de texto.
**Dados**:
- 1612-1615: França (azul-real, flor-de-lis)
- 1615-1641: Portugal (verde, esfera armilar)
- 1641-1644: Holanda (laranja, leão)
- 1644+: Portugal novamente (verde)
**Paleta**: Azul (#2C3E88) para França, Verde (#1E8449) para Portugal, Laranja (#E67E22) para Holanda
**Tamanho**: Corpo (~14×5 cm)

## 2. Prompt para IA generativa
**Prompt**: "Horizontal timeline infographic showing three colonial occupations of São Luís, Maranhão, Brazil. 1612-1615: French (blue segment with fleur-de-lis icon). 1615-1641: Portuguese (green segment with armillary sphere). 1641-1644: Dutch (orange segment with lion). 1644 onwards: Portuguese again (green). Clean modern design, each segment clearly labeled with dates and national symbol. Title: 'São Luís: three colonial powers, one city.'"
