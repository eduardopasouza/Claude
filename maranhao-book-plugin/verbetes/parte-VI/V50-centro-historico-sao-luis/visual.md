# Visual V50-A: Anatomia de um sobrado azulejado

## 1. Briefing para designer
**O que é**: Infográfico em corte lateral de um sobrado típico do Centro Histórico — 2-3 andares, com setas indicando: azulejos na fachada (isolante térmico + status), mirante no topo (vigiar porto), sacada de ferro, paredes de taipa com madeira-de-lei, porão (armazém de algodão), telhas coloniais. Comparação: estado original (esquerda) vs. estado atual em ruínas (direita).
**Elementos**: Diagrama arquitetônico simplificado, dados sobre materiais, função de cada elemento. Azulejos: 312 padrões, lastro dos navios.
**Paleta**: Azul-mar (#1A3A5C), Terracota (#C45A3C), Dourado-babaçu (#D4AC0D), Branco
**Tamanho**: Página inteira

# Visual V50-B: Mapa do Centro Histórico — restaurado vs. em ruínas

## 1. Briefing para designer
**O que é**: Mapa esquemático do Centro Histórico de São Luís (220 ha), com bairros demarcados: Praia Grande (restaurada — verde), Desterro (parcial — amarelo), Portinho/Madre Deus/Apicum (em ruínas — vermelho). Pontos de interesse: Teatro Arthur Azevedo, Beco Catarina Mina, Rua Portugal, Convento das Mercês, Fonte do Ribeirão. Perímetro UNESCO (66,65 ha núcleo) destacado.
**Paleta**: Verde-mangue (#2D5A27) para restaurado, Terracota (#C45A3C) para ruínas, Dourado (#D4AC0D) para parcial
**Tamanho**: Meia página

# Visual V50-C: Timeline — do algodão à UNESCO

## 1. Briefing para designer
**O que é**: Timeline horizontal: 1615 (traçado Frias de Mesquita) → 1780-1830 (boom algodão, construção dos sobrados) → 1860s (declínio) → 1900s (êxodo dos ricos) → 1974 (tombamento IPHAN) → 1986 (tombamento estadual) → 1987 (Projeto Reviver) → 1997 (UNESCO) → 2013 (PAC) → 2026 (100+ em risco). Linha ascendente de "construção" → pico → descendente de "decadência" → tentativas de resgate.
**Paleta**: Azul-mar (#1A3A5C), Terracota (#C45A3C), Dourado (#D4AC0D)
**Tamanho**: Corpo de texto
