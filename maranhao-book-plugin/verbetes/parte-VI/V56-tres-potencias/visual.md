# Visual V56-A: Tres potencias — linha do tempo e comparacao

## 1. Briefing para designer
**O que e**: Infografico de dupla pagina mostrando as tres colonizacoes em paralelo — timeline, bandeiras, legados. Formato tipo revista Monocle ou Taschen.
**Dados**:
- Eixo temporal: 1600-1830
- Faixa 1 (azul-royal): FRANCA 1612-1615
- Faixa 2 (laranja): HOLANDA 1641-1644
- Faixa 3 (verde): PORTUGAL 1615-1822
- Para cada: duracao, motivo, heranca, personagem-chave
- Tabela comparativa (do box do texto)
- Mini-mapa: rotas de cada potencia ate Sao Luis
**Referencia de estilo**: Infografico editorial de dupla pagina, dados densos mas legiveis
**Paleta**: Azul-royal #1A237E (Franca), Laranja #E65100 (Holanda), Verde #1B5E20 (Portugal), Azul-mar #1B4965 (accent geral)
**Tamanho**: Dupla pagina (46x28 cm)

## 2. Elementos visuais
- Timeline horizontal com tres faixas coloridas (proporcoes reais de tempo)
- Bandeiras de epoca em cada faixa
- Icones: forte (FR), navio WIC (NL), igreja (PT)
- Tabela comparativa lateral
- Mini-mapa: Atlantico com setas de Paris, Amsterdam, Lisboa convergindo para Sao Luis
