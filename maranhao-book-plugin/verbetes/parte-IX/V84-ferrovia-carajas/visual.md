# Visual V84-A: O que passa vs. o que fica

## 1. Briefing para designer
**O que é**: Infográfico de duas colunas contrastando a riqueza que transita pela EFC vs. os indicadores de pobreza do Maranhão. Visual de impacto máximo.
**Dados lado esquerdo (passa)**: 177 Mt ferro/ano, US$18-19 bi em carga, lucro Vale US$24 bi (2021), navios de 400.000 t, trens de 3,3 km
**Dados lado direito (fica)**: 51,6% pobreza, menor PIB per capita, 47 mortos, pó de ferro, R$5 mi/mês em CFEM
**Paleta**: Dourado/preto para riqueza, vermelho/cinza para custo
**Tamanho**: Página inteira

# Visual V84-B: Mapa da Estrada de Ferro Carajás

## 1. Briefing para designer
**O que é**: Mapa mostrando os 892 km da EFC de Carajás (PA) a São Luís (MA), com TIs e comunidades afetadas marcadas.
**Dados**: Carajás → Marabá → Açailândia → Santa Inês → São Luís. TIs: Mãe Maria, Awá, Caru, Arariboia. Quilombola: Santa Rosa dos Pretos. Piquiá de Baixo. Ponta da Madeira.
**Paleta**: Vermelho ferrugem para trilhos, verde para TIs, preto para pontos de conflito
**Tamanho**: Meia página
