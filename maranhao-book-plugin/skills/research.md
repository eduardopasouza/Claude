# Skill: Pesquisa

Pesquisa profunda sobre um capítulo do livro. Produz material organizado com fontes verificáveis.

## Antes de começar

1. Ler `coordination.md` — entender onde o capítulo se encaixa no arco
2. Ler `playbook.md` seção 2, Etapa 1 — seguir o processo completo

## Processo

1. Delimitar escopo — listar todas as seções e subtópicos do capítulo
2. Pesquisar cada subtópico:
   - Dados oficiais (IBGE, IPEA, INPE, FUNAI, FCP, etc.)
   - Fontes acadêmicas (livros, teses, artigos)
   - Fontes primárias (documentos históricos, relatos)
   - Fontes culturais (registros etnográficos, entrevistas publicadas)
3. Para cada informação, registrar: dado + fonte completa + se foi cruzado
4. Identificar lacunas e controvérsias
5. Sugerir ângulos narrativos

## Mínimos obrigatórios

| Critério | Mínimo |
|----------|--------|
| Fontes totais | 15+ |
| Fontes acadêmicas | 5+ |
| Dados quantitativos | Com fonte e ano, cruzados quando possível |
| Citações diretas | 3+ de especialistas ou fontes primárias |
| Lacunas | Todas documentadas |

## Regras de validação

- Dado sem fonte = não usar
- Fato com 1 fonte só = marcar "a confirmar"
- Citação sem autor/obra/ano = não usar
- Wikipedia = ponto de partida, nunca fonte final
- Fontes divergentes = registrar a divergência
