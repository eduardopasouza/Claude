## 1. O PROJETO

**"Quem é o Maranhão?"** é um almanaque visual (coffee table book, 23×28cm) com 106 verbetes sobre o estado do Maranhão, Brasil. Cada verbete responde à pergunta-título de um ângulo diferente: geografia, história, cultura, economia, identidade.

**Marca**: @quemeomaranhao
**Tagline**: "A gente descobre junto."
**Tese central**: O Maranhão é um lugar que não cabe nas respostas fáceis — é ao mesmo tempo o problema e a potência, o abandono e a criação, o paradoxo e a possibilidade.

**O que NÃO é**: denúncia, panfleto, guia turístico, lamento, livro didático.

### Regras visuais
- **Fundo quente sempre** — nunca branco puro (#FFFFFF) nem cinza frio
- **YouTube**: fundo Creme (#FAF3E8) ou Carvão (#2B2B2B)
- **Reels**: fundo Carvão (#2B2B2B) — contraste no scroll
- **Estilo**: Kurzgesagt (infográficos limpos, cores vibrantes) + National Geographic (fotografia + dados)
- **Sem clichês visuais**: nada de pôr-do-sol genérico, praia paradisíaca, "povo sorridente"
- **Diversidade real**: representar indígena, negro, mestiço, ribeirinho, urbano

### Tipografia
- **Títulos**: Playfair Display Bold
- **Texto/dados em tela**: Source Sans Pro SemiBold

## 3. TOM E VOZ

### Narração
- **Tom**: literário, envolvente, irônico com a história, respeitoso com o presente
- **Inspiração**: Eduardo Bueno (coloquial, pop sem ser raso) + Eliane Brum (profundidade, humanidade)
- **Narrador**: maranhense implícito ("nossa terra", "nosso povo" — com parcimônia)
- **2ª pessoa**: interpela o espectador ("Você provavelmente nunca ouviu falar...")
- **Humor**: onde cabe (reggae, carnaval). NUNCA onde não cabe (escravidão, pobreza)

### Vocabulário obrigatório
- "Escravizado/a" (NUNCA "escravo/a")
- "Povos indígenas" (NUNCA "índios")
- "Comunidades quilombolas" (NUNCA "remanescentes", exceto contexto legal)
- Verbos no PRESENTE para povos e práticas que existem hoje

### O que NUNCA fazer
- Acusar nomes, empresas ou setores diretamente
- Romantizar pobreza nem vitimizar populações
- Usar linguagem acadêmica ("conforme demonstrado por Fulano")
- Usar linguagem turística ("terra de belezas naturais")

## 4. FORMATOS DE VÍDEO

### YouTube (8 minutos — ensaio visual)

```
HOOK (0:00-0:30): Narração + visual de impacto. Prender nos primeiros 5 segundos.
ATO 1 (0:30-2:30): Contexto + primeiro bloco narrativo
ATO 2 (2:30-5:00): Desenvolvimento + dados + comparações
ATO 3 (5:00-7:00): Clímax narrativo + revelação
FECHAMENTO (7:00-8:00): Reflexão + CTA

Elementos: narração contínua, B-roll, infográficos animados, dados em destaque,
mapas, transições suaves, trilha adequada ao tema.
```

**Thumbnail**: imagem de impacto + texto curto (2-5 palavras) em Source Sans Pro Bold branco com sombra. Logo @quemeomaranhao no canto.

### Reels/Shorts (30-90 segundos — pílula vertical)

```
HOOK (0-3s): Texto em tela + imagem/dado de impacto
DESENVOLVIMENTO (3-45s ou 3-70s): Narração + visuais alternados
SURPRESA/CTA (últimos 5-10s): Dado surpreendente + "Quem é o Maranhão? #N"

Formato vertical 9:16. Fundo Carvão. Texto grande, contraste alto.
Série numerada (#1, #2, #3...).
```

## 8. INSTRUÇÕES PARA O GEMINI

### O que produzir

Para cada vídeo piloto:
1. **Ler o youtube.md** (ou reel-N.md) do verbete — contém roteiro completo com narração, B-roll, infográficos, transições
2. **Ler o texto.md** — contém a narrativa completa com dados e fontes
3. **Ler o visual.md** — contém briefings visuais com prompts de IA generativa
4. **Gerar**: narração em áudio (PT-BR, voz masculina adulta, tom literário), imagens/ilustrações (estilo editorial, paleta do projeto), infográficos animados (Kurzgesagt-style), transições, trilha sonora adequada
5. **Montar**: vídeo final editado seguindo o roteiro

### Regras de produção
- **Paleta obrigatória**: usar as cores hex definidas acima
- **Nunca usar branco puro** (#FFFFFF) como fundo
- **Logo**: @quemeomaranhao (Playfair Display Bold) visível em todo vídeo
- **Fontes dos dados**: sempre visíveis em rodapé
- **Sem imagens de pessoas reais** sem autorização — usar ilustrações/animações
- **Estilo de ilustração**: editorial, cores quentes, traço pictórico (não fotorrealista)
- **Trilha**: adequada ao tema. Reggae para V69. Percussão para cultura. Instrumental suave para dados.

### Entregáveis por vídeo
- Vídeo editado (MP4, 1080p mínimo)
- Thumbnail (1280×720, JPG)
- Legendas (PT-BR .srt)
- Descrição para YouTube (com timestamps, fontes, tags)

## 9. CONTEXTO EDITORIAL — O QUE O GEMINI PRECISA SABER

1. **O autor é advogado do agronegócio no Maranhão.** O livro protege essa posição: dados frios, nunca acusações diretas. O vídeo deve seguir a mesma regra.

2. **Três matrizes com peso igual.** Indígena, africana e europeia recebem tratamento idêntico. Nunca privilegiar uma sobre outra.

3. **Ironia com a história, neutralidade com o presente.** Eventos coloniais podem ser tratados com ironia (o absurdo dos impérios). Temas contemporâneos recebem dados — o espectador conclui.

4. **"Escravizado", não "escravo".** Sempre. Sem exceção.

5. **Dados falam por si.** O vídeo apresenta dados, contexto e comparação. Nunca conclui acusando.

6. **O epílogo-bússola** (`epilogo-bussola.md`) é o calibrador de tom para todo o projeto. Ler antes de produzir.
