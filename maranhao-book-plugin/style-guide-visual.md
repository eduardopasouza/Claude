# Style Guide Visual — Quem é o Maranhão?

> Documento de referência para toda produção visual do projeto: livro, YouTube, Reels, redes sociais. Define paleta, tipografia, grid, identidade de boxes, infográficos, mapas, briefings visuais e templates de vídeo.
> Versão: 2.0 | Data: 2026-03-17 | Alinhado com coordination-v3.md

## 1. IDENTIDADE VISUAL — PRINCÍPIOS

### Conceito

O Maranhão é um estado de **contrastes**: cerrado e mangue, seca e chuva, riqueza cultural e pobreza material, patrimônio e abandono. A identidade visual traduz isso:

- **Calor e terra** como base (paleta quente)
- **Modernidade e clareza** no tratamento (estilo infográfico)
- **Rigor e acessibilidade** na informação (dados bonitos E precisos)
- **Identidade reconhecível** em qualquer plataforma

### Referências visuais

| Referência | O que capturar |
|-----------|----------------|
| **Kurzgesagt** | Infográficos limpos, cores vibrantes sobre fundo quente, animações explicativas |
| **Vox Borders / Vox Atlas** | Mapas narrativos, sobreposição de dados em paisagens, storytelling visual |
| **DK "As Grandes Ideias"** | Layout modular de livro-referência, boxes destacados, hierarquia visual clara |
| **National Geographic (livros)** | Fotografia + infográfico + texto integrados, mapas de alta qualidade |
| **The Pudding** | Visualização de dados criativa, interatividade, narrativa guiada por dados |

### O que NÃO é

- Não é turístico (sem filtros tropicais, sem saturação exagerada)
- Não é acadêmico (sem visual árido, sem gráficos Excel)
- Não é institucional (sem azul-governo, sem logotipos oficiais como elemento central)
- Não é nostálgico (sem sépia generalizado, sem "era bom antigamente")

## 2. PALETA DE CORES

### Cores primárias

| Cor | Nome | Hex | Uso principal |
|-----|------|-----|---------------|
| 🟫 | **Ocre** | `#C8952E` | Cor-base, fundos, títulos. Representa a terra, o babaçu, o cerrado |
| 🟤 | **Terracota** | `#B5533E` | Destaques, acentos, boxes. Representa o barro, a cerâmica, o calor |
| 🟩 | **Verde-mata** | `#2D6A4F` | Natureza, dados positivos, Amazônia maranhense |
| 🟦 | **Azul-mar** | `#1B4965` | Água, litoral, mapas, dados neutros |

### Cores secundárias

| Cor | Nome | Hex | Uso |
|-----|------|-----|-----|
| 🟡 | **Areia** | `#E8D5B7` | Fundos claros, papel, áreas de respiro |
| ⚪ | **Creme** | `#FAF3E8` | Fundo principal do livro (não branco puro) |
| ⬛ | **Carvão** | `#2B2B2B` | Texto principal (não preto puro) |
| 🔴 | **Vermelho-bumba** | `#C1292E` | Alertas, dados negativos, destaque pontual. Referência ao Bumba-meu-boi |
| 🟣 | **Roxo-tambor** | `#5E3A7E` | Cultura afro-maranhense, Tambor de Crioula, religiosidade |

### Cores de fundo

| Contexto | Cor |
|----------|-----|
| Página padrão (livro) | Creme `#FAF3E8` |
| Fundo de box | Areia `#E8D5B7` com 50% opacidade |
| Fundo de infográfico | Ocre `#C8952E` com 15% opacidade |
| Fundo de vídeo YouTube | Creme `#FAF3E8` ou Carvão `#2B2B2B` (temas claro/escuro) |
| Fundo de Reel | Carvão `#2B2B2B` (contraste no scroll) |

### Regras de uso

- **Fundo quente**: sempre. Nunca branco puro (#FFFFFF) nem cinza frio
- **Contraste mínimo**: WCAG AA (4.5:1 para texto, 3:1 para elementos gráficos)
- **Máximo 3 cores por composição**: 1 dominante + 1 acento + 1 neutro
- **Dados negativos**: vermelho-bumba. **Dados positivos**: verde-mata. **Dados neutros**: azul-mar
- **Cada Parte tem cor de acento própria** (dentro da paleta), criando identidade visual por seção

### Paleta por Parte (11 Partes)

**1ª metade — cronológica:**

| Parte | Nome | Cor de acento | Justificativa |
|-------|------|--------------|---------------|
| I | O Chão | Verde-mata `#2D6A4F` | Terra, biomas, natureza, o palco antes do homem |
| II | Os Primeiros | Terracota `#B5533E` | Povos originários, terra, ancestralidade |
| III | A Conquista | Azul-mar `#1B4965` | Chegada pelo mar, navios, colonização |
| IV | O Povo Negro | Roxo-tambor `#5E3A7E` | Travessia, resistência, cultura afro-maranhense |
| V | Apogeu e Queda | Ocre `#C8952E` | Ouro branco (algodão), riqueza e declínio |

**2ª metade — temática:**

| Parte | Nome | Cor de acento | Justificativa |
|-------|------|--------------|---------------|
| VI | São Luís: A Ilha | Azul-mar `#1B4965` | Ilha, baía, azulejos, insularidade |
| VII | O Povo e a Identidade | Terracota `#B5533E` | Mestiçagem, identidade, formação do povo |
| VIII | As Criações | Vermelho-bumba `#C1292E` | Festa, música, boi, criação cultural |
| IX | A Economia | Ocre `#C8952E` | Produção, riqueza, ciclos econômicos |
| X | O Estado e a Estrutura | Carvão `#2B2B2B` (acento: Vermelho-bumba) | Poder, dados duros, desigualdade |
| XI | O Maranhão no Mundo | Verde-mata `#2D6A4F` | Futuro, sustentabilidade, potencial |

**Nota**: Cores de acento se repetem entre Partes — isso é intencional. A distinção entre Partes é reforçada pela combinação cor + posição no livro + conteúdo visual. Partes com mesma cor de acento nunca são adjacentes.

## 3. TIPOGRAFIA

### Famílias tipográficas

| Uso | Família | Estilo | Fallback |
|-----|---------|--------|----------|
| **Títulos (livro)** | Playfair Display | Bold | Georgia, serif |
| **Subtítulos** | Playfair Display | Regular / Italic | Georgia, serif |
| **Texto corrido (livro)** | Source Serif Pro | Regular 400 | Georgia, serif |
| **Boxes e glossário** | Source Sans Pro | Regular 400 | Helvetica, sans-serif |
| **Dados e infográficos** | Source Sans Pro | SemiBold 600 | Helvetica, sans-serif |
| **Números grandes** | Playfair Display | Bold | Georgia, serif |
| **Vídeo (tela)** | Source Sans Pro | SemiBold 600 | Helvetica, sans-serif |

> **Nota**: todas são fontes Google Fonts, gratuitas e open-source. Decisão final de tipografia pode mudar com diagramação profissional.

### Hierarquia tipográfica (livro)

```
PARTE (Playfair Display Bold, 36pt, cor de acento da Parte, caixa alta)
  TÍTULO DO VERBETE (Playfair Display Bold, 24pt, carvão)
    Subtítulo / Seção (Playfair Display Italic, 16pt, cor de acento)
      Texto corrido (Source Serif Pro Regular, 11pt, carvão)
        Nota de rodapé (Source Serif Pro Regular, 8pt, carvão 70%)
```

### Hierarquia tipográfica (vídeo)

```
TÍTULO DO VÍDEO (Source Sans Pro Bold, grande, branco sobre fundo escuro)
  SUBTÍTULO (Source Sans Pro SemiBold, médio, cor de acento)
    DADO EM DESTAQUE (Playfair Display Bold, muito grande, cor de acento)
      LEGENDA (Source Sans Pro Regular, pequeno, branco)
        FONTE (Source Sans Pro Light, muito pequeno, branco 70%)
```

## 6. INFOGRÁFICOS

### Princípios

1. **Bonito E preciso** — estética Kurzgesagt, dados reais com fonte
2. **Autoexplicativo** — entendível sem ler o texto ao redor
3. **Fonte sempre visível** — rodapé do infográfico, Source Sans Pro 7pt
4. **Paleta do projeto** — nunca cores fora da paleta definida
5. **Acessível** — não depender só de cor para diferenciar (usar forma, textura, rótulo)

### Tipos de infográfico

#### Linha do tempo

```
┌──────────────────────────────────────────────────┐
│  TÍTULO DA LINHA DO TEMPO                        │
│                                                  │
│  1612 ──●── 1615 ──●── 1641 ──●── 1755 ──●──   │

│    francesa   portuguesa           do Comércio   │
│                                                  │
│  Fonte: [autor, ano]                             │
└──────────────────────────────────────────────────┘
```

- Fundo: Creme
- Linha: Carvão
- Marcos: Círculos na cor de acento da Parte
- Rótulos: Source Sans Pro 9pt

#### Mapa temático

```
┌──────────────────────────────────────────────────┐
│  TÍTULO DO MAPA                                  │
│                                                  │
```

## 7. IMAGENS E ILUSTRAÇÕES

### Categorias de imagem

| Categoria | Fonte | Tratamento |
|-----------|-------|------------|
| **Paisagens contemporâneas** | Fotos reais (banco de imagens, fotógrafos locais) | Cor natural, sem filtro exagerado |
| **Cultura viva** | Fotos reais (Bumba-meu-boi, Tambor de Crioula, festas) | Cor vibrante natural |
| **Cenas históricas** | Documentos reais (mapas, gravuras) + complemento IA | Estilo compatível com originais |
| **Retratos históricos** | Ilustração IA baseada em descrições | Estilo pictórico, não fotorrealista |
| **Mapas antigos** | Reprodução de originais + anotações modernas | Sobreposição sutil |
| **Infográficos/dados** | Produção própria (IA ou design) | Estilo Kurzgesagt |

### Tratamento de imagens históricas

- **Gravuras e mapas originais**: manter aspecto original, adicionar moldura sutil na cor de acento
- **Complemento IA**: quando não há registro visual, gerar ilustração no estilo pictórico (não fotorrealista)
- **Nunca**: apresentar imagem IA como se fosse foto real ou documento histórico
- **Sempre**: creditar a origem (ex: "Ilustração baseada em descrição de [fonte]" ou "Mapa original: [acervo]")

### Regras de imagem

1. **Sem clichês visuais**: nada de pôr-do-sol genérico, praia paradisíaca sem contexto, "povo sorridente"
2. **Contexto sempre**: toda imagem tem legenda explicativa (não só descritiva)
3. **Diversidade**: representar a diversidade real do Maranhão (indígena, negro, mestiço, ribeirinho, urbano)
4. **Dignidade**: nenhuma imagem que reduza pessoas a estereótipos ou a objetos de pena
5. **Fonte**: toda foto/ilustração creditada

### Prompt de estilo para IA (imagens do livro)

Ao gerar imagens com IA para o livro, usar como base de prompt:

```
Estilo: ilustração editorial, cores quentes (ocre, terracota, verde-mata),
traço pictórico (não fotorrealista), fundo creme/areia,
composição limpa com espaço para texto.
Referência: National Geographic editorial illustrations meets Kurzgesagt warmth.
NÃO: fotorrealismo, saturação excessiva, filtros tropicais, clichês visuais.
```

## 10. ACESSIBILIDADE

### Contraste

- Todo texto sobre fundo atende WCAG AA (4.5:1 mínimo)
- Infográficos não dependem só de cor (usar forma, textura, rótulo)
- Legendas de vídeo com fundo semi-transparente para legibilidade

### Texto alternativo

- Toda imagem no e-book tem alt text descritivo
- Infográficos têm versão textual acessível
- Vídeos têm legenda em PT-BR e EN

### Tamanho mínimo de texto

| Contexto | Mínimo |
|----------|--------|
| Texto corrido (livro) | 11pt |
| Boxes | 10pt |
| Notas de rodapé | 8pt |
| Fonte de infográfico | 7pt |
| Legenda de vídeo | 24px equivalente |

## 11. BRIEFING VISUAL — COMO PREENCHER

Cada verbete gera um `visual.md` com briefings para toda imagem necessária. O template está no `playbook-v2.md` (Skill: VISUAL).

### Checklist do briefing

```
☐ Tipo de imagem definido (foto, ilustração, infográfico, mapa)
☐ Descrição detalhada do conteúdo
☐ Dados exatos com fonte (se infográfico)
☐ Referência visual (se houver)
☐ Paleta indicada (conforme Parte)
☐ Texto na imagem (se houver)
☐ Uso definido (livro, YouTube, Reel, todos)
☐ Crédito previsto
```

### Critérios de qualidade

- Briefing específico o suficiente para produzir sem ambiguidade
- Dados conferidos com fonte
- Paleta dentro do style guide
- Sem clichês visuais (ver seção 7)
