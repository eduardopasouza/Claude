# Quem é o Maranhão?

Almanaque visual do Maranhão — 106 verbetes com dados e alma.

## Iniciar sessão de produção

1. Ler `protocolo-producao.md` (regras de comportamento)
2. Verificar `registro-producao.md` (onde parou)
3. Seguir pipeline: Pesquisa → Outline → [Aprovação] → Texto → Reels → Visual
4. Atualizar registros ao final de cada verbete

## Arquivos principais

| Arquivo | Função |
|---------|--------|
| `protocolo-producao.md` | **Ler primeiro.** Regras de produção sequencial, pipeline, filas, checklists. |
| `foundation.md` | Fundamentação teórica: premissas, referencial, fatos estruturantes, teses por Parte. |
| `vision.md` | Premissas editoriais, decisões de design, expectativas quantitativas. |
| `coordination-v3.md` | Índice completo dos 106 verbetes, visão, regras, pipeline. |
| `playbook-v2.md` | Manual de produção: 10 skills com templates, exemplos e checklists. |
| `proposta-editorial-v2.md` | Proposta editorial formal. |
| `style-guide-editorial.md` | Guia de estilo editorial: voz, tom, vocabulário, regras. |
| `style-guide-visual.md` | Identidade visual: paleta, tipografia, grid, boxes, vídeo. |
| `registro-producao.md` | Dashboard de produção: verbetes concluídos, reels, visuais. |
| `changelog.md` | Registro cronológico de toda produção e decisão. |
| `epilogo-bussola.md` | Ensaio-epílogo (calibrador de tom). |
| `registro-decisoes-73-rodadas.md` | Histórico de 296 decisões em 74 rodadas. |
| `banco-dados/dados-centrais.yaml` | Banco central de dados factuais (160+ entradas). |
