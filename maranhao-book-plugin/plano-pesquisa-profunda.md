## Regras de Pesquisa

1. **Uma fonte por vez**: pesquisar devagar, ler o conteúdo real, extrair dados verificáveis
2. **URL obrigatória**: toda informação extraída deve ter URL ou referência bibliográfica completa
3. **Grounding**: o dado no texto deve corresponder EXATAMENTE ao dado na fonte
4. **Divergência**: se a fonte oficial diverge do texto, corrigir o texto
5. **Timestamp**: registrar data de acesso de cada URL
6. **Banco YAML**: todo dado verificado alimenta o banco imediatamente
