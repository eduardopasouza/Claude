# Guia de Pesquisa — Quem é o Maranhão?

> Documento normativo. Define como a pesquisa é conduzida, quais fontes são obrigatórias, como catalogar referências e como validar informações. Nenhum tópico é redigido sem passar por este processo.

---

## 1. PRINCÍPIO FUNDAMENTAL

**Nada se escreve sem pesquisa. Nada.**

Cada tópico do livro (800-1.200 palavras) exige pesquisa prévia com fontes reais catalogadas. O texto só começa depois que a pesquisa está documentada no banco de dados centralizado.

```
PESQUISA → FICHAMENTO → OUTLINE → REDAÇÃO
   ↑                                    |
   └── Se faltam fontes, volta aqui ────┘
```

## 3. HIERARQUIA DE FONTES

```
NÍVEL 1 (máxima confiabilidade):
  Fontes primárias (documentos de época)
  Dados oficiais (IBGE, FUNAI, INPE, etc.)

NÍVEL 2 (alta confiabilidade):
  Livros acadêmicos publicados por editoras reconhecidas
  Artigos em periódicos com revisão por pares
  Teses e dissertações defendidas

NÍVEL 3 (confiabilidade moderada):
  Historiografia clássica (verificar com fontes recentes)
  Reportagens investigativas (Folha, Estadão, Intercept, Repórter Brasil)
  Documentários com créditos de pesquisa

NÍVEL 4 (confiabilidade baixa — usar com cuidado):
  Jornalismo diário
  Canais de YouTube especializados
  Entrevistas publicadas

NÍVEL 5 (nunca como fonte final):
  Wikipedia — APENAS como ponto de partida para localizar fontes reais
  Posts em redes sociais
  Blogs sem credenciais
```

**Regra de ouro**: subir sempre na hierarquia. Se encontrou um dado na Wikipedia, vá à fonte que a Wikipedia cita. Se a Wikipedia não cita fonte, buscar em nível 1 ou 2.

## 4. PROCESSO DE PESQUISA POR TÓPICO

### Passo 1: Definir escopo

Antes de pesquisar, responder:
- Qual é o **argumento central** deste tópico?
- Quais **afirmações factuais** preciso fundamentar?
- Quais **dados numéricos** preciso encontrar?
- Quais **citações diretas** enriqueceriam o texto?
- Este tópico tem **conexão direta** com quais outros?

### Passo 2: Busca por categoria

| Ordem | Ação |
|-------|------|
| 1º | Verificar se o **banco de dados centralizado** já tem fontes úteis |
| 2º | Buscar **dados oficiais** para qualquer afirmação numérica |
| 3º | Buscar **historiografia** para contexto e narrativa |
| 4º | Buscar **artigos/teses** para análises e interpretações recentes |
| 5º | Buscar **fontes primárias** quando o tópico é histórico |
| 6º | Pesquisa web ativa: Google Scholar, sites de dados, repositórios |

### Passo 3: Fichamento no banco de dados

Cada fonte encontrada é catalogada imediatamente (ver Seção 5).

### Passo 4: Relatório de pesquisa

Para cada tópico, produzir um **relatório breve** antes de redigir:

```markdown
# Pesquisa — Tópico X.Y: [Título]

## Argumento central
[1-2 frases: o que este tópico defende/mostra]

## Fontes encontradas
- [id-fonte-1]: [o que oferece para este tópico]
- [id-fonte-2]: [o que oferece para este tópico]
- ...

## Dados-chave (com fonte)
- [dado 1] — Fonte: [id]
- [dado 2] — Fonte: [id]

## Citações diretas selecionadas
- "[citação]" — [Autor], [Obra], p. XX

## Lacunas
- [O que não encontrei e precisaria]
- [Dado sem fonte confiável]
- [Divergência entre fontes]

## Conexões cruzadas
- Tópico X.Z: [como se conecta]
- Tópico W.V: [como se conecta]
```

**Saída**: integrado no `research.md` do capítulo (`drafts/cap-XX-titulo/research.md`).

## 6. REGRAS DE VALIDAÇÃO

### 6.1 Regras absolutas

| Situação | Ação |
|----------|------|
| Dado numérico sem fonte | **NÃO USAR** — buscar fonte ou remover |
| Fato histórico com 1 fonte só | Marcar como **"a confirmar"** — buscar segunda fonte |
| Citação sem autor/obra/ano | **NÃO USAR** |
| Wikipedia como fonte | **NÃO** — ir à fonte que a Wikipedia cita |
| YouTube como fonte única | **NÃO** — verificar com fonte acadêmica/oficial |
| Fonte jornalística como único suporte | Aceitável para fatos recentes; para dados históricos, **buscar corroboração** |

### 6.2 Tratamento de divergências

Quando fontes divergem sobre um fato ou dado:

1. **Registrar a divergência** no banco de dados (campo `notas`)
2. **Citar ambas** no texto via nota de rodapé:
   > ² Meireles (2001, p. 55) indica 1612 como data da fundação; Lacroix (2008, p. 23) sugere que a cerimônia ocorreu no início de 1613.
3. **Não escolher uma** arbitrariamente — apresentar as duas versões
4. Se possível, indicar qual tem mais suporte documental
5. Se a divergência é significativa, pode virar um box "Você Sabia?"

### 6.3 Dados que precisam de atualização

Dados do IBGE, IPEA, INPE etc. mudam com frequência. Regras:

- Usar sempre o **dado mais recente disponível**
- Registrar **data de acesso** no banco de dados
- No texto, indicar o **ano do dado** (não da publicação):
  > "Em 2022, o Maranhão tinha 7,1 milhões de habitantes.¹"
- Para séries históricas, indicar o **período**: "entre 2010 e 2022, a população cresceu X%"
- Se o dado pode ter mudado significativamente, adicionar nota:
  > ³ Dados do Censo 2022. Estimativas mais recentes podem diferir.

## 7. PESQUISA WEB ATIVA

### 7.1 Regra geral

O Claude **deve buscar fontes reais na web** durante a produção de cada tópico. Não se limitar ao que sabe — ir atrás do que existe.

### 7.2 O que buscar por tipo de tópico

| Tipo de tópico | Buscar |
|----------------|--------|
| **Histórico** (caps 2-7) | Historiografia + fontes primárias digitalizadas + teses recentes |
| **Indígena** (cap 8) | FUNAI + teses de antropologia + ISA (Instituto Socioambiental) |
| **Socioeconômico** (caps 9-10, 19-22) | IBGE + IPEA + PNAD + Atlas + DataSUS + SNIS |
| **Cultural** (caps 11-16) | IPHAN + teses de antropologia/musicologia + registros do Patrimônio |
| **Geográfico** (cap 1, 17-18) | IBGE Mapas + INPE + MapBiomas + ICMBio |
| **Ambiental** (1.10, desmatamento) | INPE PRODES + MapBiomas + teses de ecologia |
| **Futuro** (caps 23-24) | Dados recentes + planos governamentais + pesquisas de tendência |

### 7.3 Estratégia de busca

```
1. Busca no Google Scholar: "[tema] Maranhão"
2. Busca no site do IBGE/IPEA para dados numéricos
3. Busca no repositório UFMA para teses locais
4. Busca no SciELO para artigos brasileiros
5. Busca geral para reportagens investigativas relevantes
6. Busca em acervos digitais para fontes primárias
```

### 7.4 O que fazer quando NÃO encontra

1. Registrar a lacuna no relatório de pesquisa
2. Indicar ao autor que a informação não tem fonte confiável
3. **Nunca inventar** — se não tem fonte, não entra no livro
4. Sugerir ao autor formas de preencher a lacuna (ex: contatar pesquisador, consultar acervo físico)

## 8. CRITÉRIOS DE QUALIDADE DA PESQUISA

### 8.1 Mínimos por tópico

| Critério | Mínimo |
|----------|--------|
| Fontes totais | 3+ (sendo ao menos 1 acadêmica ou primária) |
| Categorias de fonte | 2+ das 4 categorias |
| Dados numéricos | Com fonte, ano e nota de rodapé |
| Citações diretas | 1+ de especialista ou fonte primária |
| Lacunas documentadas | Todas listadas no relatório |

### 8.2 Mínimos por capítulo (soma dos tópicos)

| Critério | Mínimo |
|----------|--------|
| Fontes totais | 15+ |
| Fontes acadêmicas | 5+ |
| Fontes oficiais (dados) | Para todo dado numérico |
| Citações diretas | 3+ |
| Referências cruzadas mapeadas | 2+ para outros capítulos |

### 8.3 Checklist de pesquisa

Antes de passar para o outline/redação:

```
☐ Argumento central definido
☐ Fontes catalogadas no banco de dados centralizado
☐ Mínimo 3 fontes por tópico (2+ categorias)
☐ Todo dado numérico com fonte e ano
☐ Citações diretas selecionadas (com autor, obra, página)
☐ Divergências entre fontes documentadas
☐ Lacunas identificadas e registradas
☐ Conexões cruzadas mapeadas
☐ Relatório de pesquisa produzido
```
