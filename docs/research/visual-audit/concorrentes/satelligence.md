## Propósito declarado
"Sustainability Monitoring Simplified." A Satelligence é uma empresa holandesa de monitoramento satelital que ajuda corporações (tradings, varejo global, bancos) a garantir **sourcing livre de desmatamento e conversão**, com foco pesado em compliance EUDR (regulamento europeu anti-desmatamento) e reporte de Scope 3 / NDPE / CSRD.

## Features principais observadas
- **Compliance EUDR:** kit para demonstrar que commodities (café, cacau, soja, óleo de palma, madeira, borracha, gado) importadas para UE são livres de desmatamento pós-2020.
- **NDPE (No Deforestation, Peatland, Exploitation):** monitoramento para cadeias de óleo de palma e florestais.
- **V-DCF (Verified Deforestation and Conversion Free):** validação de sourcing livre de desmatamento e conversão (cerrado, pampas).
- **Scope 3 / CSRD / GHGp:** mensuração de emissões land-based para reporte corporativo europeu.
- **Mapa satelital com dados de eventos EUDR** (alertas de desmatamento ligados a propriedades/fornecedores).
- **Grievance resolution workflow** — investigar denúncia, checar evidência satelital, responder rápido.
- Conhecimento externo: 95% de acurácia em validação de CAR/polígonos, uso de SAR (radar) para atravessar nuvens, série temporal Landsat + Sentinel + Planet.

## UX / interações
Produto logado é plataforma de **analytics + mapa** com "real-time insights & reporting". Fluxo típico do usuário corporativo: upload da lista de fornecedores → Satelligence cruza com polígonos/propriedades → gera mapa de risco por fornecedor → alerta quando há desmatamento detectado na área de influência → relatório exportável para auditor EUDR. Trabalha por cadeia de commodity, não por propriedade individual.

## Preço e modelo de negócio
Sem preço público. B2B enterprise global — clientes são traders (Cargill, ADM, Bunge), varejo europeu, bancos com carteira de commodities, certificadoras. Modelo anual + taxa por volume de fornecedores/polígonos monitorados. Conhecimento externo: ticket a partir de EUR 50-100k/ano para cliente médio, podendo chegar a milhões para grandes traders.

## Autenticação
Plataforma logada em app.satelligence.com (link "Log in" no header e footer). Autenticação presumível via email+senha corporativo com SSO disponível para clientes enterprise. Não tem login público/self-service.

## Insights para AgroJus
1. **Fazer diferente:** Satelligence é ferramenta do COMPRADOR europeu para acusar cadeia. AgroJus pode ser ferramenta do PRODUTOR/advogado brasileiro para responder a essas acusações — lado oposto do pipeline EUDR. Mercado fala isso como necessidade real.
2. **Copiar:** estratégia de 3 pilares (Compliance / Sustainable Sourcing / Scope 3) cada um alinhado a um framework regulatório. AgroJus pode fazer pilares por regulação: Código Florestal / Lei 9.605 / EUDR / Res. CMN 4.327.
3. **Copiar:** uso de SAR (radar) para cobertura em nuvem. Amazônia brasileira tem 60%+ de cobertura de nuvem grande parte do ano — diferencial real.
4. **Copiar:** credenciais de confiança no footer (ISO 27001, auditorias externas). AgroJus deve buscar LGPD + ISO + auditoria quando escalar.
5. **Fazer diferente:** Satelligence esconde produto atrás de demo comercial. AgroJus deve ter tela real do produto no marketing.
6. **Copiar:** workflow de "grievance resolution" (investigar denúncia → checar evidência → responder). É exatamente o fluxo do advogado respondendo a auto de infração.
7. **Copiar:** posicionar dado satelital como EVIDÊNCIA legal, não só indicador — importante em peça jurídica.
8. **Fazer diferente:** Satelligence é global em inglês. AgroJus é Brasil-first em PT; não tentar competir globalmente cedo.
