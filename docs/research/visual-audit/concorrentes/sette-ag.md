# Sette

- **URL:** https://sette.ag/
- **Categoria:** concorrente
- **Data auditoria:** 2026-04-17

## Propósito declarado
"Rapidez e segurança em toda a jornada do financiamento agrícola." A Sette entrega uma plataforma end-to-end de crédito rural — do território e análise ambiental ao contrato, assinatura e registro de garantias. É o concorrente mais ambicioso deste painel em escopo de jornada.

- **Main:** três linhas de produto empilhadas — **Rural Uno** (plataforma integrada), **Ativus** (assinatura/registro/certidões), **CPR Sette** (verificação de CPR em 7 passos).

## Features principais observadas
### Rural Uno (6 motores)
- **Motor Territorial:** identificação de propriedade por CPF/CNPJ (matrícula + CAR + georref).
- **Motor Agro:** imagens de satélite com IA para identificação de cultura e segmentação de talhões (Deep Learning + redes convolucionais).
- **Motor ESG:** avaliação de compliance ambiental (embargos, desmatamento, sobreposição com UC/TI).
- **Motor Financeiro:** análise de capacidade de crédito via DRE do produtor.
- **Monitoramento Satelital:** monitoramento contínuo em tempo real.
- **Rural Vis:** dashboard com alertas personalizados para decisão ágil.

### Ativus
- **Assinador digital:** qualificado com autenticação via Biometria Facial / SMS / WhatsApp.
- **Registro de garantias.**
- **Busca e leitura automatizada de certidões** (usa LLMs).
- **Consultas** diversas (fiscal, protesto, etc).

### CPR Sette
- Fluxo de 7 passos para emissão e verificação de CPR (Cédula de Produto Rural).

## UX / interações
O menu "Soluções" é estruturado por **ação** do ciclo de crédito (buscar proprietário → analisar risco → emitir certidão → assinar → registrar garantia → monitorar). Padrão dominante é "Entre em contato" (venda consultiva), mas o dashboard Rural Vis sugere produto logado com painel próprio. A declaração explícita de stack IA (Deep Learning, CNN, Transformers, LLMs, Machine Learning, árvores de decisão) é incomum no mercado — a Sette se posiciona como tech-forward. UX provável do produto: busca por CPF/CNPJ → mapa com talhões segmentados por IA → painel ESG/Financeiro → geração e assinatura de CPR.

## Preço e modelo de negócio
Sem preço. Modelo B2B enterprise vendido para bancos, tradings, cooperativas e agentes de crédito rural. Três linhas permitem venda modular ou pacote completo (Rural Uno + Ativus + CPR Sette = jornada inteira). Conhecimento externo: ticket esperado na faixa R$ 10-80k/mês dependendo da linha e volume, com taxa adicional por consulta/CPR registrada.

## Autenticação
No produto Ativus a **assinatura digital** usa autenticação por **Biometria Facial / SMS / WhatsApp** (assinatura qualificada conforme MP 2.200-2 / ICP-Brasil com alternativa de atributo biométrico). Para o portal do cliente, presumível email/senha + 2FA (não detalhado).

## Conhecimento externo aplicável
- Sede em Londrina/PR, filiais SP e Manaus — DNA do agronegócio paranaense.
- Conglomerado de produtos construído por aquisição + desenvolvimento interno (CPR Sette e Ativus parecem ter origens distintas unificadas).
- Clientes: cooperativas (Sicredi, Sicoob), bancos regionais, tradings.
- Concorre com Softfocus (em MCR), Serasa/Traive (em análise) e Agrotools (em geomonitoramento) — diferencial é consolidar as três na mesma stack.

## Insights para AgroJus
1. **Copiar:** arquitetura de "motores" dentro de plataforma (Motor Territorial / Motor Agro / Motor ESG / Motor Financeiro). AgroJus pode ter Motor Ambiental / Motor Fundiário / Motor Creditício / Motor Tributário — vocabulário técnico que comunica capacidade.
2. **Copiar:** menu estruturado por AÇÃO do fluxo (buscar → analisar → emitir → assinar → registrar). AgroJus deve ter menu ação-first (Buscar fazenda → Analisar risco → Montar defesa → Protocolar → Monitorar).
3. **Fazer diferente:** Sette vende jornada do CRÉDITO. AgroJus pode vender jornada do LITÍGIO (triagem → análise → pesquisa → redação → protocolo → monitoramento), que é o que mIA já faz.
4. **Copiar:** assinatura digital qualificada com biometria — crítica para peças assinadas pelo cliente no app.
5. **Copiar:** Rural Vis como dashboard com alertas personalizados. AgroJus precisa de "AgroJus Vis" com alertas de publicações DJ, novas multas, novos embargos.
6. **Copiar:** declarar stack IA explicitamente (CNN, Transformers, LLMs). Para investidor/advogado premium sinaliza profundidade técnica.
7. **Fazer diferente:** Sette resolve a jornada DO BANCO. AgroJus resolve a jornada do ADVOGADO DO PRODUTOR — quando o banco nega, erra ou é omisso.
8. **Copiar:** busca automatizada + leitura de certidões com LLM. Combina com pipeline de ingestão de documentos já em uso (mia-project).
